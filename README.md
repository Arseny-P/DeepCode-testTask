## Использованные технологии

- Архитектура: FSD
- React
- TypeScript
- UI-библиотека: Consta
- Стейт-менеджер: Zustand
- Форматтер Prettier и линтер ESLint
- Система сборки: Vite
- Для HTTP-запросов: TanStack Query, Axios

## Что было реализовано?

Небольшое приложение с получение данных о пользователях и их постах.

> Обеспечена стабильная обработка состояний загрузки и ошибок на всех страницах приложения

### Авторизация и навигация

- Вход по access-токену с проверкой на заполнение
- Выбор между просмотром постов или пользователей

### Работа со списками

- Вывод данных о пользователях (Имя, Фамилия, Email) и постах (ID, заголовок)
- Навигация «Следующая/Предыдущая», переключение по номерам страниц и настройка лимита (10/25/50 строк)
- Переход к детальному просмотру по клику на строку

### Детальные карточки

На каждой карточке есть кнопка, для перехода назад на главную страницу

- Карточка поста
    - Название поста
    - ID поста
    - Автор поста
    - Полный текст поста
    - Все комментарии к посту
- Карточка пользователя
    - Заглушка аватарки с инициалами пользователя
    - Имя пользователя
    - ID пользователя
    - Статус 
    - Email пользователя
    - Гендер
